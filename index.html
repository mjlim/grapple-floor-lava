Grapple Tag: The Floor is Lava

tentative title. I felt like trying to get back into gamedev again so I'm having a go using physicsjs.

concept: 2-4 player local multiplayer arena game with the following constraints:

  1. everyone has a grappling hook that can be used to pull themselves towards walls or players towards themselves
  2. the floor is lava

inspirations: starwhal, samurai gunn

current status (may 8): working on gamepad support. next is having multiple players in the space at once as well as a better test level
